2. Write jQuery code to find all h1 elements that are children of a div element and make their background color red.

Sample HTML: <body> <h1>abc</h1><br> <div> <h1>div-1</h1> <h1>div-2</h1> </div> <h1>xyz</h1> </body> My answer: <script> $(document).ready(function(){ $("div p").css("background-color","red"); }); </script>

3. Use a jQuery method to insert the text "YES!" at the end of the

<p> element.

My answer: <script> $(document).ready(function(){ $("p").prepend("YES!"); }); </script>
